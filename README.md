# Слайдер для сайта

Автономный слайдер проектов ремонта, реализованный на чистом JavaScript без использования сторонних библиотек. Все элементы слайдера генерируются динамически из единого массива данных.

## Функциональность

- ✅ Переключение слайдов стрелками влево/вправо с кольцевой логикой
- ✅ Переключение слайдов по точкам (индикаторам)
- ✅ Переключение слайдов по ссылкам в навигационном меню
- ✅ Единая структура данных для слайдов
- ✅ Универсальная функция переключения слайдов
- ✅ Динамическая генерация всех элементов из массива данных
- ✅ Поддержка клавиатуры (стрелки влево/вправо)

## Использование

1. Откройте `index.html` в браузере
2. Используйте стрелки, точки или ссылки для навигации
3. Для добавления нового слайда просто добавьте объект в массив `slidesData` в `script.js`

## Добавление нового слайда

Для добавления нового слайда достаточно добавить объект в массив `slidesData`:

```javascript
const slidesData = [
    {
        city: 'Rostov-on-Don, Admiral',
        area: '81 m2',
        time: '3.5 months',
        cost: 'Upon request',
        image: './img/rostov_admiral.jpg'
    },
    // Добавьте новый объект здесь
    {
        city: 'Moscow, Central',
        area: '120 m2',
        time: '5 months',
        cost: 'Upon request',
        image: './img/moscow_central.jpg'
    }
];
```

При добавлении нового объекта автоматически создаются:
- Новая навигационная ссылка в меню
- Новый слайд с изображением
- Новая точка-индикатор

## Технические детали

### Архитектура

- **Чистый JavaScript** (без библиотек и фреймворков)
- **CSS Grid** с использованием `grid-area` для структуры макета
- **Динамическая генерация** всех элементов из массива данных
- **Отсутствие хардкода** в HTML — только контейнеры для динамического контента

### Структура макета

- Навигационное меню сверху отдельным блоком
- Левая колонка: информация о проекте (COMPLETED PROJECTS, описание, детали)
- Правая колонка: изображения слайдов
- Колонки делят экран пополам и выровнены по верхней линии

### Структура данных

Каждый слайд содержит:
- `city` — название города и района
- `area` — площадь квартиры
- `time` — время ремонта
- `cost` — стоимость ремонта
- `image` — путь к изображению
